## ***How to Install?***
***Windows:***
```bash
git clone https://github.com/sancho1952007/Modern-GUI-v3.0.git
cd Modern-GUI-v3.0
python main.pyw
```

<br>

***Linux:***
```bash
git clone https://github.com/sancho1952007/Modern-GUI-v3.0.git
cd Modern-GUI-v3.0
python3 main.pyw
```
<br>

***To Use Custom Code, Edit The `main.py` File.***

<br>
